
DESCRIPTION
===========

The Quora Pixel module provides integration with the Quora Pixel,
allowing you to place a single pixel across your entire website to report
conversions.

The module handles the following Quora Pixel events:

  - Generic 
  - Complete Registration 
  - Add Payment Info
  - Add to Cart
  - Add to Wishlist
  - Search 
  - Initiate Checkout
  - Purchase


INSTALLATION
============

1. Enable Module
2. Change event code for tracking different conversion types
  a. <script>qp('track', '<conversion item to track>');</script>
  b. example:
    -   <script>qp('track','<Generic>');</script>
